Homework #6: Conclusion

Quasi-experimental designs for learning systems

In summary, we developed a causal inference framework, with specific application to learning health systems, for measuring the direct impact that intervening according to a risk prediction algorithm has on a patient’s outcome. Our major finding is that through a regression discontinuity design under local randomization, we can treat patients that fall inside a window as-if randomly assigned to treatment, which can be used to assess those interventions made according to a risk prediction model and to eliminate challenges associated with working with observational data. Moreover, this framework can be naturally extended to multi-cutoff and muti-score regression discontinuity designs to reflect the learning and adapting of a learning health system. Unfortunately, we were unable to apply this framework to real data from a learning health system and draw inferences about the results obtained. However, we were able to create synthetic data and validate our method. While we mainly focused on the estimation, identification, and the sensitivity of the “no interference scenario”, possible avenues for future work include the further development and application of this framework when interference between units is allowed, as well as the implementation of this framework on publicly available risk data and ultimately real hospital data. This work is expected to contribute not only in the learning health systems scenario, but also in any other application in which a risk prediction model is used, and has the potential to become the standard framework for evaluating the effect of intervening according to a risk model.
